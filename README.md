# Otus project

Project for MLOps course

Setup Instructions
1. Clone the repository:
    ```bash
    git git@github.com:lizkakostereva-lgtm/otus-project.git
    cd otus-project
    ```
2. Create a project:
    ```bash
    uv python pin 3.11
    uv init
    uv venv
    uv add --requirements requirements.txt
    ```
